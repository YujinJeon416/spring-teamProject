# spring-teamProject
스프링 프레임워크를 이용한 팀프로젝트 (영화예매사이트)  
20.07.22 ~ 20.08.13(23일)  


## 역할 분담(나의 역할)
* 스토어 홈페이지  
* 푸드 상품 구매   
* 푸드 상품 바로 구매  
* 장바구니 기능  
* 관리자(푸드 상품 관리)  
* 관리자(푸드매출)  
* 마이페이지 푸드 구매 내역  
* PPT 정리
